About

This plugin provides matching and verification of Avro serialised messages to the Pact contract testing framework. It requires a version of the Pact framework that supports the V4 Pact specification as well as the Pact plugin framework.

Supported Pact framework versions:

  • Pact-JVM v4.4.x

  • Pact-Rust Consumer v0.9.x

  • Pact-Rust Verifier v0.9.x

  • Pact-Go v2.0.0-beta
